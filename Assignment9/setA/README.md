# Que1

void settime(int, int, int) to set the specified values of hours, minutes and seconds in object </br>
void showtime() to display contents of time object </br>

time diff(time) subtract values of hours, minutes and seconds in time object argument from current time object after making appropriate conversions and return time difference </br>
Write a main program to illustrate the use of above class. </br>
